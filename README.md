## Run Locally

Clone the project

```bash
  git clone https://github.com/ozgeahras/unit-converter.git
```

Go to the project directory

```bash
  cd ..unit-converter
```

Install dependencies

```bash
npm install
```

Start the app

```bash
npm run start
```
